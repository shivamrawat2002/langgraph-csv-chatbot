# 📊 LangGraph CSV / Excel AI Chatbot

Chat with your CSV and Excel files using AI — powered by LangGraph + Groq (FREE).

---

## ⚡ Setup (3 Steps)

### 1. Install requirements
```bash
pip install -r requirements.txt
```

### 2. Add your free API key
Open the `.env` file and replace with your key:
```
GROQ_API_KEY=your_groq_api_key_here
```
Get your FREE key at → https://console.groq.com (no credit card needed)

### 3. Run the app
```bash
streamlit run app.py
```

Opens in your browser automatically at `http://localhost:8501`

---

## 💡 How to Use

1. Upload any **CSV or Excel** file from the sidebar
2. Type your question in plain English
3. Get instant AI answers about your data!

## 🧪 Example Questions
- "How many rows are in this data?"
- "What is the average salary?"
- "Which product had the highest sales?"
- "Are there any missing values?"
- "Show me summary statistics"
